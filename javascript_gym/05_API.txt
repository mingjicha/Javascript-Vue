✔ Application Programming Interface : API란❓ 
    응용 프로그램에서 사용할 수 있도록 
    운영체제나 프로그래밍 언어가 제공하는 기능을 
    제어할 수 있게 만든 인터페이스
    ex) 컴퓨터 메모장에 쓰는 인터페이스면 키보드가 되고 
        쓴 것을 읽는 인터페이스는 모니터가 된다. - 물리적인 하드 UI
    사용자가 사용하는 인터페이스 User Interface (UI)
        1) 인터페이스 : 소통을 하기 위한 접점
        2) UI : 사용자가 소통하기 위한 접점
        3) API : 응용 프로그램에서 소통하기 위한 접점

    💡응용 프로그램은 운영체제에서 실행되는 모든 소프트웨어 = 애플리케이션
        - API : 애플리케이션에서 데이터를 읽거나 쓰기 위해서 사용하는 인터페이스 
            🧐앱에서 소통하기 위해 사용하는 인터페이스

HTTP API 
    1) 기상청⛅
        기상청에서 API를 제공해주면 app에서는 curl을 요청해서 응답받음 📍05_API1.png, 05_API2.png 참고 
            1-1) 기상청에서는 동네 날씨 정보를 DB서버에 저장함 📍05_API3.png 참고
            1-2) 날씨와 관련된 데이터를 다른 애플리케이션에서 가져갈 수 있도록 서버에 다양한 API를 개발 : 🎨프론트엔드
            1-3) 날씨 정보 애플리케이션에서 이러한 API를 조회하여 사용자에게 보여질 UI를 개발 : 💻백엔드
    2) 배달앱🛵 📍05_API4.png
        사용자는 UI를 사용하여 소통하고 애플리케이션은 API를 사용하여 소통
            2-1) 주문량이 쌓이고 마케팅 팀에서 양념치킨을 가장 많이 먹는 성별대와 연령대를 볼 수 있는 웹 페이지를 개발해달라고 함
            2-2) 쌓인 데이터에서 통계를 추출하여 프론트엔드가 애플리케이션을 호출 할 수 있도록 API를 개발 : 💻백엔드
            2-3) 이 API를 사용하여 마케팅 팀에게 보여질 웹 UI를 개발 : 🎨프론트엔드

💼회사 자체에서 서비스를 만들기 위해 사내에서 개발하고 사용하는 API를 Private API라고 하며 제3자에게 공개되지 않음.
📤누구나 사용할 수 있게 개방해놓은 API는 Public API 도는 Open API라고 함
    ex) 공공데이터, 유튜브, 인스타, 페이스북, 네이버, 카카오
        - 많은 플랫폼에서 자사 데이터나 기능을 사용할 수 있도록 다양한 API를 제공 👉HTTP API(프로토콜 = 소통방법 = 통신방법)
