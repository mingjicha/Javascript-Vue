반복문 : 동일한 작업을 여러번 반복

1. for
for(let i = 0; i < 10; i++) {
  반복할 코드
}
let i = 0; i < 10; i++ 에서 ;로 구분하며 세부분으로 나눌 수 있음
-> let i = 0; 초기값 설정(반복문을 진행할 때 처음 한번만 실행)
-> i < 10; 조건(반복문을 돌면서 조건을 확인하고 false가 되면 멈춤/true라면 실행)
-> i++ 코드 실행 후 작업(반복문 한 번 실행된 후 해야할 작업을 명시)
=> i = 0이고 i가 10보다 작으므로 코드를 실행하고 i값은 1씩 증가를 반복

2. while
let i = 0;
  while(i < 10) {
    코드
  }

3. do.. while
let i = 0;
do {
  코드
  i++;
} while (i < 10)
✅while과 차이점, 적어도 한 번은 코드를 실행함

✅반복문을 빠져나오는 기능
1. break : 멈추고 빠져나옴(만나는 순간 즉시 코드 실행을 멈추고 해당 반복문을 빠져나옴)
2. continue : 멈추고 다음 반복으로 진행(코드 실행을 멈추고 반복문을 빠져나오진 않고 다음 반복으로 점프)

💡명확한 횟수가 정해져 있으면 for문 아니면 while, 그에 비해 do..while은 잘 사용 안함
